Apa itu Module Informatics?

Tidak banyak mahasiswa saat ini kebingungan saat mencari referensi, baik itu untuk menambah pengetahuannya maupun untuk memenuhi tugas dari Dosen. Website ini akan menjawab masalah tersebut.

Website ini dibuat untuk para mahasiswa khususnya mahasiswa Teknik Informatika Universitas Singaperbangsa Karawang dengan tujuan untuk wadah sebagai media Modul - modul yang berkaitan dengan Teknik Informatika.

Didalamnya terdapat beberapa modul yang bisa di download dan di upload oleh mahasiswa itu senditi supaya bisa dipelajari oleh mahasiswa lain. Dan ada juga beberapa tutorial mengenai pemrograman.

Website ini menggunakan bahasa Markup Hypertext atau HTML, CSS, PHP dan Javascript. Hasil website cukup membantu para mahasiswa dalam menambah skill, terutama bagi bagi kelas yang mempunyai dosen yang sama sehingga bisa berbagi modul dan catatan
